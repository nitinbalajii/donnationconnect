# DonationConnect

DonationConnect is a web application designed to coordinate donations between individuals, shelter homes, and volunteers. It provides a centralized platform for managing the logistics of item and food donations across local communities.

## Features

- Donor Dashboard: Allows users to view current shelter needs, submit donation requests, and track their delivery status.
- Shelter Dashboard: Enables shelter administrators to list required items, accept incoming donations, and update their facility details.
- Volunteer System: Provides a delivery interface for volunteers to claim and complete donation transport tasks.
- Authentication & Access Control: Custom dashboards and secure routing based on the registered user's role (Donor, Shelter, or Volunteer).
- Responsive Architecture: Built with a mobile-first approach using CSS Grid and Flexbox for cross-device compatibility.

## Technical Stack

Frontend: HTML, CSS, Vanilla JavaScript
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
Authentication: JSON Web Tokens (JWT)

## Repository Structure

- Root Directory: Frontend assets including HTML views, CSS styling, and client-side JavaScript.
- /backend: Node.js API server, encompassing database models, routing logic, and middleware.
